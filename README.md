# פרויקט בדיקות (Unit testing)

---------
#מגיש
### רז יעקבי

זהו פרויקט Java המכיל אוסף של פונקציות עזר (Utility Methods) לביצוע פעולות נפוצות על מספרים, מחרוזות ורשימות. הפרויקט כולל חבילת בדיקות יחידה (Unit Tests) מקיפה באמצעות **JUnit 5** ו**Gradle**

## תיאור הפונקציונליות

המחלקה `App` כוללת את המתודות הבאות:

### מתמטיקה
* `add(int a, int b)`: חיבור שני מספרים שלמים.
* `isPrime(int n)`: בדיקה האם מספר הוא ראשוני.
* `factorial(int n)`: חישוב עצרת של מספר (זורק `IllegalArgumentException` עבור מספרים שליליים).
* `fibonacciUpTo(int limit)`: החזרת סדרת פיבונאצ'י עד לגבול מסוים.
* `average(int[] nums)`: חישוב ממוצע של מערך מספרים.

### מניפולציות על מחרוזות
* `reverse(String s)`: הפיכת סדר התווים במחרוזת.
* `isPalindrome(String s)`: בדיקה האם מחרוזת היא פלינדרום (מתעלמת מסימני פיסוק ואותיות גדולות/קטנות).
* `isAnagram(String s1, String s2)`: בדיקה האם שתי מחרוזות הן אנגרמות.
* `charFrequency(String s)`: חישוב תדירות הופעה של כל תו במחרוזת.
* `mostCommonWord(String s)`: איתור המילה הנפוצה ביותר בטקסט (מתעלם מסימני פיסוק ורגישות לאותיות).

### עיבוד רשימות
* `filterEvens(List<Integer> nums)`: סינון והחזרת המספרים הזוגיים בלבד מתוך רשימה.

## דרישות מערכת
* **Java Development Kit (JDK)** 8 ומעלה.
* **Gradle** (לניהול הבנייה והרצת הבדיקות).
